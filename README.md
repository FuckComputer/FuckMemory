## Installation
### Dependencies
- make
- g++
### Linux & macOS
Just use

```shell
sudo make install
```

then,  `FuckMemory` will automatically be installed in `/usr/bin/fuckmem`.

### Windows
Use g++ in cmd to compile `src/main.cpp` and run it.

## Uninstallation
### Linux & macOS
Use

```shell
sudo make uninstall
```

and FuckMemory will be deleted from your computer.
